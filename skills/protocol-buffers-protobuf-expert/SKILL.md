# `protocol-buffers-protobuf-expert`
Serialização de alta performance para a era moderna.
## Regras
1. Defina mensagens em arquivos `.proto` com campos numerados de forma imutável (não mude os números após o lançamento).
2. Utilize tipos escalares apropriados para minimizar o tamanho do payload serializado.
3. Siga as convenções de style guide do Google para nomes de pacotes, mensagens e campos.
