# `long-term-memory-storage-logic`
Dando uma biografia e continuidade histórica para as suas IAs.
## Regras
1. Extraia e armazene "Entidades" e "Preferências" importantes de cada conversa em um banco de dados relacional ou de grafos.
2. Utilize o padrão "Summarized History": ao invés de guardar tudo, guarde os pontos chaves que definem a relação com o usuário.
3. Garanta que o usuário possa visualizar e apagar sua "Memória de Longo Prazo" para cumprir leis de privacidade (LGPD/GDPR).
