# `skeleton-loading-pattern`
Reduz a ansiedade do usuário durante o carregamento de dados.
## Regras
1. O esqueleto deve imitar fielmente a estrutura final do componente.
2. Adicione uma animação de "shimmer" (brilho) para indicar atividade.
3. Não mostre o esqueleto por menos de 200ms para evitar flickering.
