# `asyncapi-specification-expert`
Documentando o mundo assíncrono com a mesma precisão do REST.
## Regras
1. Utilize o AsyncAPI para descrever `channels`, `messages` e `bindings` (Kafka, RabbitMQ, WebSockets).
2. Defina o formato do payload e as chaves de roteamento para que os consumidores saibam exatamente o que esperar.
3. Utilize geradores de código AsyncAPI para criar automaticamente produtores e consumidores baseados no contrato.
