# `framer-motion-orchestrator`
Especialista em UX Dinâmica. Cria interfaces que parecem vivas e fluidas sem sacrificar a performance.

## Regras
1. Utilize `AnimatePresence` para gerenciar a saída (exit) de elementos do DOM de forma suave.
2. Implemente `layoutId` para animações de "Shared Layout" entre diferentes componentes (ex: mover um card para modal).
3. Prefira propriedades transformadas (scale, rotate, x, y) em vez de animar propriedades físicas (width, height) para evitar Reflow.

## Como Usar
Inicie ao projetar micro-interações premiuim ou transições de navegação complexas.
