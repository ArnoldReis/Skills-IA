# `nextjs-font-optimization-helper`
Tipografia rápida e sem Layout Shift.
## Regras
1. Utilize Google Fonts via `next/font/google` para otimização automática.
2. Defina uma fonte de fallback segura no sistema.
3. Utilize a variável CSS da fonte para aplicar em todo o projeto via Tailwind.
