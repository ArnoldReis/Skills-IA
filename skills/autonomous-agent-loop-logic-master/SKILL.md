# `autonomous-agent-loop-logic-master`
A espinha dorsal de agentes que trabalham de forma independente.
## Regras
1. Mantenha uma "Memória de Curto Prazo" das ações anteriores para evitar que o agente entre em loops infinitos.
2. Defina uma "Mission Statement" clara que o agente deve consultar em cada iteração do loop.
3. Implemente um limite máximo de passos (Max Iterations) para garantir que o agente não gaste recursos infinitamente.
