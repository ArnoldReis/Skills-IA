# `polyglot-persistence-strategy-pro`
O banco de dados certo para cada tipo de pergunta.
## Regras
1. Utilize Bancos Relacionais para transações complexas e consistência forte.
2. Utilize Bancos de Grafos para relacionamentos profundos e redes sociais.
3. Utilize Bancos de Documentos ou Chave-Valor para dados de alta volumetria e esquemas flexíveis.
