# `lgpd-privacy-by-design-expert`
Privacidade como padrão, não como uma camada adicional.
## Regras
1. Colete apenas os dados estritamente necessários para a finalidade pretendida (Minimização de Dados).
2. Implemente o "Direito ao Esquecimento" facilitando a deleção completa de dados de um usuário sob demanda.
3. Obtenha consentimento explícito e granular para diferentes finalidades de uso de dados.
