# `react-lazy-load-image-pro`
Economia de banda e carregamento inicial veloz.
## Regras
1. Use `loading="lazy"` nativo quando possível.
2. Para situações complexas, use scroll listener ou Intersection Observer.
3. Adicione efeito de fade-in quando a imagem termina de carregar.
