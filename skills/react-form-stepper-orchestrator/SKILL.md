# `react-form-stepper-orchestrator`
Fluxos complexos divididos em partes digeríveis.
## Regras
1. Bloqueie a avançada para o próximo passo se o atual for inválido.
2. Permita ao usuário voltar e editar passos anteriores.
3. Forneça uma barra de progresso visual.
