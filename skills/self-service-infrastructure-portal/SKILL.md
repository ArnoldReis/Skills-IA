# `self-service-infrastructure-portal`
Autonomia com governança para equipes de desenvolvimento.
## Regras
1. Ofereça "Golden Paths" (templates pré-aprovados) para recursos comuns como bancos de dados e filas.
2. Garanta que todas as permissões e políticas de segurança sejam injetadas automaticamente nos recursos criados.
3. Monitore o custo da infraestrutura provisionada via portal para evitar desperdícios.
