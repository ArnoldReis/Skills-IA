# `sbom-software-bill-of-materials-pro`
O inventário completo de tudo o que compõe o seu produto.
## Regras
1. Gere arquivos SBOM em formatos padrão como CycloneDX ou SPDX em cada build de produção.
2. Compartilhe o SBOM com clientes corporativos que exigem conformidade rigorosa de segurança.
3. Utilize o SBOM para realizar triagens rápidas quando uma nova vulnerabilidade zero-day (ex: Log4j) for anunciada.
