# `grafana-dashboard-design-pro`
Visualização estratégica para decisões operacionais rápidas.
## Regras
1. Coloque os indicadores mais importantes (Disponibilidade, Erros, Latência) no topo do dashboard (The Four Golden Signals).
2. Utilize cores de forma semântica: Verde para bom, Amarelo para atenção, Vermelho para crítico.
3. Adicione links para logs de rastreamento (Traces) e documentação (Runbooks) diretamente nos gráficos.
