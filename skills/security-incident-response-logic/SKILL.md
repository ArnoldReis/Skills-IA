# `security-incident-response-logic`
Velocidade e ordem no caos das brechas de segurança.
## Regras
1. Isole imediatamente recursos comprometidos (Quarentena) assim que um alerta de alta gravidade for confirmado.
2. Preserve evidências (dumps de memória, snapshots de disco) para análise forense posterior.
3. Automatize a notificação das partes interessadas (DPO, Diretoria, Clientes) baseando-se na gravidade do vazamento.
