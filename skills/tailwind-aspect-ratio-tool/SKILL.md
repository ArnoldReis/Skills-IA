# `tailwind-aspect-ratio-tool`
Imagens e vídeos que não distorcem em diferentes telas.
## Regras
1. Utilize utilitários nativos como `aspect-video` ou `aspect-square`.
2. Forneça fallbacks para navegadores antigos se necessário.
3. Combine com `object-cover` para garantir preenchimento total do container.
