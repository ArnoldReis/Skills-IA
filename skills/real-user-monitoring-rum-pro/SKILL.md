# `real-user-monitoring-rum-pro`
Monitoramento do ponto de vista de quem realmente usa o sistema.
## Regras
1. Utilize Web Vitals (LCP, FID, CLS) para medir a performance de renderização no navegador do usuário.
2. Analise a performance baseada em diferentes tipos de conexão (4G, Wi-Fi) e dispositivos (Desktop, Mobile).
3. Respeite as leis de privacidade (LGPD/GDPR) anonimizando IPs e dados sensíveis capturados pelo sensor de RUM.
