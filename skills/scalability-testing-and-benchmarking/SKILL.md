# `scalability-testing-and-benchmarking`
Certificando que o sistema está pronto para o próximo milhão de usuários.
## Regras
1. Utilize ferramentas de teste de carga (ex: k6, Gatling) para simular tráfego exponencial e identificar pontos de quebra.
2. Realize "Step Tests" onde a carga aumenta gradualmente para observar o comportamento do auto-scaling.
3. Compare o desempenho entre diferentes arquiteturas através de benchmarks controlados e reprodutíveis.
