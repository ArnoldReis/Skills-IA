# `data-loss-prevention-dlp-logic`
Prevenindo o vazamento de informações confidenciais por descuido ou dolo.
## Regras
1. Utilize expressões regulares (Regex) e algoritmos de soma de verificação (Luhn) para identificar números de cartões e documentos.
2. Intercepte tráfego de saída (Egress) em busca de padrões de dados sensíveis não criptografados.
3. Alerte a equipe de segurança sempre que uma expetativa de dados em massa for detectada.
