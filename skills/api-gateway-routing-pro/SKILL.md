# `api-gateway-routing-pro`
Porta de entrada única para o seu ecossistema.
## Regras
1. Encaminhe requisições com base em prefixos de path ou subdomínios.
2. Centralize autenticação e rate limiting no gateway para aliviar os microserviços.
3. Implemente Circuit Breakers para evitar o efeito cascata em serviços lentos.
