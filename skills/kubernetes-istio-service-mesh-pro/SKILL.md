# `kubernetes-istio-service-mesh-pro`
Controle total sobre a comunicação entre microserviços.
## Regras
1. Utilize Mutual TLS (mTLS) para criptografar todo o tráfego interno.
2. Configure VirtualServices para realizar deploys Canary ou Blue/Green.
3. Monitore métricas de telemetria nativas do Istio no Kiali.
