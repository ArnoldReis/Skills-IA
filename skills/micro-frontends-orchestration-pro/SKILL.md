# `micro-frontends-orchestration-pro`
Independência mútua para grandes aplicações web.
## Regras
1. Escolha uma técnica de integração: Composição no Servidor (SSI), Build-time ou Runtime (Module Federation).
2. Mantenha um "Container" ou Shell que carrega e orquestra os micro-frontends dinamicamente.
3. Garanta que o Design System seja compartilhado para manter a consistência visual entre pedaços independentes.
