# `identity-and-access-management-iam-pro`
Uma fonte única da verdade para todas as identidades na organização.
## Regras
1. Implemente Single Sign-On (SSO) para que os usuários utilizem uma única credencial em todos os sistemas da empresa.
2. Utilize protocolos padrão como OIDC para integração de identidade entre aplicações.
3. Automatize o provisionamento e desprovisionamento (SCIM) de contas para evitar "usuários fantasmas".
