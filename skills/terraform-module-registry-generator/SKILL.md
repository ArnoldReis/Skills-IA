# `terraform-module-registry-generator`
Módulos de infraestrutura como blocos de montar.
## Regras
1. Utilize versionamento semântico (SemVer) em cada módulo.
2. Forneça exemplos (`examples/`) completos de uso para cada módulo.
3. Inclua testes automatizados via Kitchen-Terraform ou Terratest.
