# `browser-notification-api-wrapper`
Engajamento direto no desktop do usuário.
## Regras
1. Solicite permissão explicitamente antes de tentar enviar notificações.
2. Utilize ícones e badges consistentes com a marca.
3. Não envie notificações excessivas ou intrusivas.
