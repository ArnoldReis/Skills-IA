# `load-balancing-algorithms-pro`
Equanimidade e eficiência na distribuição de requisições.
## Regras
1. Utilize "Least Connections" para serviços onde as requisições têm tempos de processamento muito variados.
2. Utilize "IP Hash" para garantir afinidade de sessão quando o estado da aplicação não for totalmente distribuído.
3. Implemente o "Weighted Round Robin" para levar em conta servidores com diferentes capacidades de hardware.
