# `ai-self-correcting-code-logic`
Software que se cura sozinho através de ciclos de feedback.
## Regras
1. Capture e forneça o erro completo (Stack Trace) para a IA quando uma execução de código falhar.
2. Permita que a IA adicione logs (print debugging) temporários para entender o estado das variáveis em tempo de execução.
3. Force a criação de testes unitários automatizados para validar que a "correção" não quebrou outras partes do sistema.
