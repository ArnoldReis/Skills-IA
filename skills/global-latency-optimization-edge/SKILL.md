# `global-latency-optimization-edge`
Perto do usuário em qualquer lugar do planeta.
## Regras
1. Execute lógica de autenticação e redirecionamento na "Borda" (Edge) antes mesmo de chegar ao backend.
2. Utilize "Regional Database Clusters" para manter os dados fisicamente próximos da origem da requisição.
3. Minimize o número de "Round Trips" de rede através de protocolos modernos como HTTP/3 (QUIC).
