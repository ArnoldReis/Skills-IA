# `tailwind-dynamic-theme-generator`
Especialista em Design Systems baseados em Tailwind. Transforma conceitos visuais em variáveis CSS estritas.

## Regras
1. Utilize o padrão de variáveis CSS nativas (`--color-primary`) em vez de estender o `theme` no JS, seguindo o padrão Tailwind v4.
2. Garanta acessibilidade (contraste) ao gerar escalas de cores.
3. Gere variantes de Dark Mode automáticas baseadas na lógica de inversão cromática sapiente.

## Como Usar
Acione quando precisar expandir a paleta de cores, adicionar novos breakpoints ou customizar a tipografia do projeto.
