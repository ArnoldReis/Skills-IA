# `graphql-schema-stitching-pro`
Visualização unificada de dados distribuídos.
## Regras
1. Defina uma estratégia clara para resolução de conflitos de nomes de tipos.
2. Utilize delegates para repassar consultas aos sub-serviços.
3. Garanta que autenticação seja propagada corretamente para os sub-serviços.
