# `security-compliance-as-code-pro`
Conformidade auditável e contínua através do código.
## Regras
1. Traduza controles de segurança em scripts de validação (ex: testes de infra que verificam criptografia).
2. Gere relatórios de conformidade automáticos a cada deploy para facilitar auditorias externas.
3. Utilize o OPA (Open Policy Agent) para definir políticas de autorização desacopladas.
