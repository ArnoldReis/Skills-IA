# `cloud-budget-alert-automator`
Controle financeiro proativo no Cloud.
## Regras
1. Crie alertas em 50%, 80% e 100% do valor orçado mensalmente.
2. Envie notificações imediatas para canais de FinOps ou gerentes de projeto.
3. Utilize previsões (Forecasted spend) para alertar antes que o limite seja atingido.
