# `security-log-aggregation-pro`
Visão holística da postura de segurança em tempo real.
## Regras
1. Padronize os logs usando o formato ECS (Elastic Common Schema) para facilitar a correlação.
2. Identifique sequências de eventos que sozinhos parecem inofensivos, mas juntos indicam um ataque.
3. Alerte em tempo real sobre picos de falhas de autenticação em múltiplos serviços simultâneos.
