# `container-security-hardening-pro`
Tornando seus contêineres ambientes hostis para atacantes.
## Regras
1. Utilize imagens base mínimas (ex: Alpine ou Distroless) para reduzir pacotes desnecessários e superfície de ataque.
2. Execute seus processos como usuários não-root dentro do contêiner.
3. Escaneie as imagens em busca de vulnerabilidades antes de enviá-las para o seu Registry privado.
