# `waf-web-application-firewall-logic`
A barreira de entrada contra tráfego malicioso e bots.
## Regras
1. Ative regras gerenciadas para bloquear SQLi, XSS e vulnerabilidades conhecidas de frameworks.
2. Utilize "Geoblocking" para restringir acesso de países onde você não possui operação comercial se o risco for alto.
3. Monitore os "Falsos Positivos" para evitar que usuários legítimos sejam bloqueados por regras excessivamente agressivas.
