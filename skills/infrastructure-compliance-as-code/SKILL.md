# `infrastructure-compliance-as-code`
Segurança auditável desde o início do desenvolvimento.
## Regras
1. Integre o scanner no pipeline de Pull Request.
2. Bloqueie o merge se existirem falhas de segurança de nível 'High' ou 'Critical'.
3. Siga frameworks padrão como CIS Benchmarks ou SOC2.
