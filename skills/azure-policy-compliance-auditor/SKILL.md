# `azure-policy-compliance-auditor`
Governança e conformidade na nuvem Microsoft.
## Regras
1. Bloqueie a criação de recursos fora das regiões (locais) permitidas.
2. Garanta que todos os recursos tenham as tags de 'projeto' e 'dono'.
3. Audite se bancos de dados SQL Azure estão com encriptação habilitada.
