# `react-drag-and-drop-helper`
Interfaces interativas e lúdicas.
## Regras
1. Utilize a API nativa de Drag-and-Drop ou bibliotecas como `dnd-kit`.
2. Forneça feedbacks visuais claros (dash borders, opacity) durante o arraste.
3. Garanta acessibilidade através de controles alternativos via teclado.
