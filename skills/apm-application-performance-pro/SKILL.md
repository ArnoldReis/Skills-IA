# `apm-application-performance-pro`
Raio-X da execução do código em produção.
## Regras
1. Monitore o tempo de execução de cada rota de API e identifique os gargalos de CPU ou memória.
2. Rastreie automaticamente todas as queries SQL geradas pel ORM para encontrar consultas lentas ou o problema N+1.
3. Utilize instrumentação de código para medir a performance de bibliotecas críticas de processamento.
