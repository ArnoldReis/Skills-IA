# `gcp-cloud-run-deploy-helper`
Containers Serverless escaláveis sem esforço.
## Regras
1. Utilize o modo de execução 'sem servidor' para economia de custo em ociosidade.
2. Configure limites de instâncias (mínimo e máximo) para gerenciar o escalonamento.
3. Ative a integração com Cloud SQL via sockets para melhor performance.
