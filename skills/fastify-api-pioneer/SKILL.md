# `fastify-api-pioneer`
Base para servidores Node.js modernos e escaláveis.
## Regras
1. Utilize `register` para modularizar plugins e rotas.
2. Defina schemas JSON para Inputs e Outputs para acelerar a serialização.
3. Utilize hooks de ciclo de vida (`onRequest`, `preHandler`) para segurança e logging.
