# `tailwind-typography-pro`
Design editorial de alta qualidade no web.
## Regras
1. Utilize o sistema de `prose` para renderização de Markdown e textos longos.
2. Defina tamanhos de fonte baseados em escalas harmônicas (ex: Major Third).
3. Garanta espaçamento entre linhas (leading) confortável para leitura.
