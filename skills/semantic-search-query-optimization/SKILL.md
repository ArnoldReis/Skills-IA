# `semantic-search-query-optimization`
Encontrando a agulha no palheiro através do significado.
## Regras
1. Utilize modelos de "Embedding" de alta qualidade que capturem as nuances específicas do domínio da sua aplicação.
2. Implemente a busca "Híbrida" (Semântica + Keyword BM25) para os casos onde termos técnicos precisos são fundamentais.
3. Refine a consulta do usuário original (Query Expansion) para incluir sinônimos e conceitos relacionados antes de vetorizar.
