# `runtime-application-self-protection-rasp`
Imunidade ativa integrada ao sistema.
## Regras
1. Intercepte chamadas de banco de dados e arquivos para verificar se o payload foi adulterado.
2. Bloqueie ataques de injeção mesmo se as defesas perimetrais (WAF) falharem.
3. Alerte com contexto rico: mostre exatamente qual linha de código foi alvo do ataque.
