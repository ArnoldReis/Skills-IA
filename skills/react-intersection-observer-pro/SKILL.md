# `react-intersection-observer-pro`
Reactividade baseada na posição de scroll.
## Regras
1. Utilize para disparar animações de entrada apenas quando visível.
2. Desligue a observação após o primeiro gatilho se não for mais necessária (Trigger Once).
3. Gerencie múltiplas referências de forma performática.
