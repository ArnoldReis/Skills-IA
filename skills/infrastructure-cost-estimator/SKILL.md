# `infrastructure-cost-estimator`
Visibilidade financeira antes do deploy.
## Regras
1. Utilize ferramentas como Infracost para ler o arquivo de plano JSON.
2. Identifique recursos "Zumbi" que geram custos sem utilidade.
3. Compare custos entre diferentes tipos de instâncias cloud.
