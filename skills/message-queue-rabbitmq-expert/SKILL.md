# `message-queue-rabbitmq-expert`
O poder do roteamento flexível e entrega confiável de mensagens.
## Regras
1. Utilize `Exchanges` (Direct, Topic, Fanout) para rotear mensagens de forma inteligente baseada em metadados.
2. Configure a persistência de mensagens e filas para garantir que os dados não sejam perdidos em caso de queda do broker.
3. Utilize a confirmação de recebimento (Acknowledgements) para garantir que uma mensagem só seja removida ao ser processada com sucesso.
