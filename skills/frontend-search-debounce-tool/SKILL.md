# `frontend-search-debounce-tool`
Economiza recursos do servidor e melhora a UX.
## Regras
1. Utilize um atraso padrão de 300ms.
2. Garanta que a última chamada sempre seja executada.
3. Forneça uma opção para cancelar a busca pendente.
