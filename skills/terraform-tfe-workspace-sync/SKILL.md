# `terraform-tfe-workspace-sync`
Gestão escalável de ambientes no Terraform Cloud.
## Regras
1. Utilize tags para organizar workspaces por ambiente e aplicação.
2. Configure variáveis sensitivas diretamente via API/CLI protegida.
3. Habilite o modo de execução 'Remote' para manter o histórico centralizado.
