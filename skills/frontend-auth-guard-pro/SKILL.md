# `frontend-auth-guard-pro`
Especialista em Segurança de Acesso. Protege rotas sensíveis e gerencia o fluxo de redirecionamento para login.

## Regras
1. Utilize context ou stores para verificar o estado de autenticação antes de renderizar a página.
2. Implemente o padrão de "Public vs Private" Routes de forma desacoplada.
3. Forneça suporte para "Redirect Back" (salvar a rota que o usuário tentou acessar antes do login).

## Como Usar
Configuração inicial do sistema de autenticação ou adição de novas áreas restritas (ex: Admin Dashboard).
