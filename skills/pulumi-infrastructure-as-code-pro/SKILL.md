# `pulumi-infrastructure-as-code-pro`
O poder das linguagens modernas aplicado ao provisionamento de nuvem.
## Regras
1. Utilize Tipagem Estática (TypeScript) para capturar erros de configuração de infraestrutura em tempo de compilação.
2. Integre testes unitários de infraestrutura usando frameworks familiares como Jest ou Mocha.
3. Aproveite loops e condicionais da linguagem para criar topologias complexas de forma legível.
