# `gcp-project-security-configurator`
Proteção nativa para Google Cloud.
## Regras
1. Desabilite APIs não utilizadas para reduzir a superfície de ataque.
2. Configure VPC Service Controls para evitar exfiltração de dados.
3. Habilite o Cloud Logging para todas as ações administrativas.
