# `aws-lambda-powertools-wrapper`
Observabilidade nativa para funções Serverless.
## Regras
1. Utilize o Logger para logs estruturados em JSON.
2. Habilite o Tracer (X-Ray) para mapear chamadas externas.
3. Capture métricas customizadas para monitorar a lógica de negócio.
