# `api-documentation-swagger-pro`
Documentação viva que nunca fica desatualizada.
## Regras
1. Exponha a documentação na rota `/docs`.
2. Inclua descrições claras para todos os campos e parâmetros.
3. Forneça exemplos reais de requisição e resposta para facilitar a integração.
