# `database-query-analyzer-tool`
Inteligência artificial aplicada à performance de dados.
## Regras
1. Capture planos de execução (`EXPLAIN ANALYZE`) de queries suspeitas.
2. Sugira a criação de índices ausentes ou refatoração de Joins pesados.
3. Monitore métricas de IO e CPU por consulta.
