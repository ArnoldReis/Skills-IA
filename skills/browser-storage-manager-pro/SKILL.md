# `browser-storage-manager-pro`
Especialista em Persistência de Cliente. Gerencia dados locais com expiração automática e criptografia básica.

## Regras
1. Utilize prefixos únicos para evitar colisões entre diferentes partes da aplicação.
2. Implemente serialização automática para objetos e arrays.
3. Não armazene dados sensíveis (senhas, tokens mestre) sem criptografia secundária.

## Como Usar
Ideal para salvar preferências do usuário, estado de carrinho ou tema selecionado.
