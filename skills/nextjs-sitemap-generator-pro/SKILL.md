# `nextjs-sitemap-generator-pro`
SEO impecável para crawlers.
## Regras
1. Inclua todas as rotas públicas e dinâmicas no sitemap.
2. Defina `priority` e `changefreq` adequados para cada tipo de página.
3. Integre com o App Router usando o arquivo `sitemap.ts`.
