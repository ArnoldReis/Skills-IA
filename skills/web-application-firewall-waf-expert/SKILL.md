# `web-application-firewall-waf-expert`
A primeira linha de defesa perimetral da sua web app.
## Regras
1. Bloqueie IPs conhecidos por atividades de botnet e redes Tor se a aplicação for de baixo risco.
2. Configure inspeção profunda de pacotes (Deep Packet Inspection) para payloads JSON/XML.
3. Utilize regras de Geo-Blocking para restringir acessos a regiões geográficas não atendidas pelo negócio.
