# `backend-performance-profiling-tool`
Detetive de gargalos infraestruturais.
## Regras
1. Capture Heapsnaps para analisar uso excessivo de memória.
2. Meça o tempo de execução de blocos de código sensíveis.
3. Utilize em ambiente de staging para simular cargas reais.
