# `service-mesh-istio-fundamentals`
Infraestrutura invisível para controle total do tráfego.
## Regras
1. Utilize Sidecars (Envoy) para interceptar o tráfego de rede e aplicar políticas sem mexer no código da aplicação.
2. Implemente o plano de controle (Control Plane) para gerenciar configurações globais de Mesh de forma centralizada.
3. Utilize as métricas nativas do Mesh (Golden Signals) para visualizar o mapa de dependências e saúde do sistema.
