# `frontend-performance-mark-tool`
Precisão milimétrica na análise de performance local.
## Regras
1. Utilize `performance.mark()` para registrar marcos de tempo específicos.
2. Calcule a duração entre marcas usando `performance.measure()`.
3. Analise as medidas no console ou envie para telemetria.
