# `react-error-boundary-pro`
Previne que um erro em um componente quebre a aplicação inteira.
## Regras
1. Envolva árvores críticas (como o corpo da página) em ErrorBoundaries.
2. Forneça botões de "Tentar Novamente" que resetam o estado do componente.
3. Integre logs de erro automáticos para serviços externos.
