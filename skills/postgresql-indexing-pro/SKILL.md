# `postgresql-indexing-pro`
Consultas SQL performáticas em Tabelas com milhões de registros.
## Regras
1. Utilize índices B-Tree para buscas de igualdade e range.
2. Utilize índices GIN/GiST para buscas de texto completo (TSVector).
3. Evite excesso de índices, que podem deixar as escritas (INSERT/UPDATE) lentas.
